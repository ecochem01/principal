HOVACAL - Calibrador de vapor caliente

Descripción

HOVACAL es un generador de gas de calibración especialmente diseñado para la calibración de sistemas de monitoreo continuo de emisiones. HOVACAL genera una mezcla definida de aire y vapor de agua y también es capaz de generar diferentes mezclas de vapor de otros gases como cloruro de mercurio, cloruro de hidrógeno, amoníaco, fluoruro de hidrógeno y otros con una concentración definida.

El componente deseado está disponible como solución estándar en diferentes concentraciones.

HOVACAL consta de una unidad de suministro portátil y una unidad de evaporación. La unidad de suministro y el evaporador están conectadas a través de tubos para líquido y gas y cable eléctrico para suministro de energía y señal RTD. Para generar una concentración de vapor de agua definida, el flujo de gas se ajustará con el controlador de flujo másico incorporado y el flujo de líquido se logrará con una bomba peristáltica. Para obtener concentraciones precisas, es necesario determinar el flujo de líquido con una balanza analítica.
